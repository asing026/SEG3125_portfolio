About BrainMatch

What is BrainMatch?

BrainMatch is a simple memory card game where you flip cards and try to match pairs. It helps improve short‑term memory and focus.

Why Play?
  • Improves concentration
  • Strengthens memory
  • Fun and simple challenge
  • Good for all ages
Design Principles

The game uses basic Gestalt principles:

  • Similarity: Matching cards look alike
  • Closure: Players complete patterns mentally
  • Figure‑Ground: Cards stand out from background
  • Proximity: Cards arranged in a clear grid
Inspiration

The game is inspired by classic memory games and simple brain‑training apps.
